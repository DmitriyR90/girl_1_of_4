Да, для лучшего отображения на различных мобильных устройствах можно использовать **медиазапросы** (media queries), чтобы адаптировать стили в зависимости от ширины экрана. Таким образом, твой сайт будет выглядеть хорошо на разных устройствах — от маленьких смартфонов до больших экранов.

Вот улучшенная версия твоего кода с медиазапросами для адаптивности:

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>Document</title>
    <style>
      /* Основные стили */
      .second-paragraph {
        font-size: 39px;
        margin-bottom: 5.78%;
        width: 100%; /* 100% ширины для мобильных */
        text-align: center;
      }

      .text-wrapper {
        display: flex;
        justify-content: center;
        align-items: center;
        flex-wrap: wrap; /* Позволяет обертывать элементы при необходимости */
      }

      .hundred {
        font-size: 129px;
        line-height: 90%;
      }

      .bonus {
        font-size: 96px;
      }

      /* Медиазапрос для маленьких экранов (например, смартфоны) */
      @media (max-width: 600px) {
        .second-paragraph {
          font-size: 30px; /* Уменьшаем шрифт для маленьких экранов */
        }

        .hundred {
          font-size: 80px; /* Уменьшаем размер 100% */
        }

        .bonus {
          font-size: 70px; /* Уменьшаем размер bonus */
        }
      }

      /* Медиазапрос для экранов среднего размера (планшеты) */
      @media (max-width: 900px) {
        .second-paragraph {
          font-size: 36px; /* Уменьшаем шрифт для планшетов */
        }

        .hundred {
          font-size: 100px; /* Уменьшаем размер 100% для планшетов */
        }

        .bonus {
          font-size: 85px; /* Уменьшаем размер bonus для планшетов */
        }
      }

      /* Медиазапрос для больших экранов */
      @media (min-width: 1200px) {
        .second-paragraph {
          font-size: 39px; /* Для больших экранов можно оставить стандартный размер */
        }

        .hundred {
          font-size: 129px; /* Стандартный размер 100% для больших экранов */
        }

        .bonus {
          font-size: 96px; /* Стандартный размер bonus для больших экранов */
        }
      }
    </style>
  </head>
  <body>
    <main>
      <div class="second-paragraph">
        <p>
          <span class="text-wrapper">
            <span>get</span>
            <span class="text hundred">100%</span>
          </span>
          <span class="text bonus">bonus</span> <br />
          now
        </p>
      </div>
    </main>
  </body>
</html>
```

### Объяснение:
1. **Медиазапросы (`@media`)**:
   - Мы добавили медиазапросы для разных типов экранов:
     - **max-width: 600px** — для маленьких экранов (смартфоны).
     - **max-width: 900px** — для средних экранов (планшеты).
     - **min-width: 1200px** — для больших экранов (десктопы).
   
2. **Адаптивные размеры шрифтов**:
   - Для каждого размера экрана указаны подходящие размеры шрифтов, чтобы текст не был слишком большим на маленьких экранах и не терял в читаемости на больших.

3. **`flex-wrap: wrap;`**:
   - Эта настройка позволяет элементам внутри **.text-wrapper** перенести строки при уменьшении ширины экрана. Например, при мобильных разрешениях текст может перемещаться на новые строки для лучшего отображения.

### Как это работает:
- На маленьких экранах шрифт будет уменьшен, чтобы текст не выходил за пределы экрана.
- На более широких экранах шрифт будет увеличиваться, чтобы заполнить пространство, улучшая читаемость.
- При уменьшении ширины экрана элементы текста будут адаптироваться, и, при необходимости, переноситься на новую строку.

Таким образом, теперь твой сайт будет выглядеть красиво на всех устройствах, независимо от их размера.
